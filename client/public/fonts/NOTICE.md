# 字体资源说明

本目录中的字体用于站点外观设置里的 Web Font 预设，避免访客本机未安装字体时回退到默认字体。

## 霞鹜文楷

- 预设：`lxgw-wenkai`
- 来源：`lxgw-wenkai-webfont@1.7.0`
- 包地址：https://www.npmjs.com/package/lxgw-wenkai-webfont
- 字体文件：`lxgw-wenkai/files/lxgwwenkai-regular-subset-*.woff2`
- 引入文件：`lxgw-wenkai/lxgwwenkai-regular.css`
- 许可：字体使用 SIL Open Font License 1.1；WebFont 包使用 MIT，见 `LICENSE-OFL.txt` 和 `LICENSE-lxgw-wenkai-webfont-MIT.txt`

## jf open 粉圆 2.0

- 预设：`rounded`
- 来源：`@fontpkg/jf-openhuninn-2-0@2.0.0`
- 包地址：https://www.npmjs.com/package/@fontpkg/jf-openhuninn-2-0
- 字体文件：`openhuninn/jf-openhuninn-2.0.ttf`
- 许可：SIL Open Font License 1.1，见 `LICENSE-OFL.txt`
